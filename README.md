Primero ejecutan 
Cd AjustesDeRed
bash AjustesDeRed.sh

Aki dentro de la scrypt editamos los nombres de las redes wifi que hay en nuestro alrededor para que la 
Víctima no sospeche

luego editamos con 
nano AjustesDeRed.sh
